Hello gamers, all over the world! This game is an indie game, powered by Unity. The developer team is only one person, my name is Kozári László! But you can call me just "Laca". But talk about the game... Do you like Prince of Persia, Inside, Another World, Flashback, and this type of games? If the answer is yes, you are in luck! This game is for you.
